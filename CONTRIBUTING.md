# Contributing to Machine Theory of Mind (MToM)

Thank you for your interest in contributing to this research repository. Contributions are welcome, provided they align with the project’s scientific goals and ethical standards.

## Scope of Contributions

We welcome:
- Bug fixes and code improvements
- Documentation improvements
- Reproducibility enhancements
- Experimental extensions or ablation studies
- Theoretical clarifications or comments

We especially value contributions that improve interpretability, rigor, or reproducibility.

## Research Integrity

All contributions must:
- Preserve methodological clarity and transparency
- Avoid overstated or unsupported claims
- Maintain separation between experimental results and speculative discussion
- Respect the proof-of-concept scope of the project

If you propose changes that affect reported results, please include:
- Clear justification
- Reproducible configuration changes
- Updated documentation or analysis scripts

## Ethics and Conduct

By contributing, you agree to abide by:
- The project’s Research Ethics & Responsible Use guidelines (`ETHICS.md`)
- The Code of Conduct (`CODE_OF_CONDUCT.md`)

Contributions that violate ethical standards or community expectations may be declined.

## How to Contribute

1. Fork the repository
2. Create a feature branch
3. Make your changes with clear commit messages
4. Add or update documentation as needed
5. Open a pull request describing:
   - What was changed
   - Why it was changed
   - Any implications for results or interpretation

## Communication

Use GitHub Issues for:
- Bug reports
- Questions about methodology
- Suggestions or discussion

Please keep discussions professional, focused, and respectful.

## Licensing

By contributing, you agree that your contributions will be licensed under the same license as the project, unless explicitly stated otherwise.
